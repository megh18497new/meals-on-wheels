Meals on Wheels is a dynamic, serverless food truck website powered entirely by **Terraform** and **AWS**. It fetches a daily-updated menu from a **Google Sheet**, transforms it into JSON via **Lambda**, and serves it through a blazing fast **CloudFront CDN**. All infrastructure is built as code.‚Ä®

---

## ‚ú® Features

- ‚úÖ **Live Menu Updates** ‚Äî Edit a Google Sheet, and the menu on the website updates instantly.
- üöÄ **Fully Serverless** ‚Äî No EC2, no backend infra. Uses AWS Lambda + API Gateway + S3 + CloudFront.
- üß± **Infrastructure as Code** ‚Äî Terraform automates everything: S3, Lambda, IAM, CloudFront, API Gateway.
- ‚ö° **CDN Backed** ‚Äî The static site is served via AWS CloudFront for high availability and low latency.
- üîê **Decoupled Architecture** ‚Äî The frontend fetches live data from a secure API, not bundled files.
- üß™ **Minimal Stack** ‚Äî Built with pure HTML, CSS, JavaScript (no frameworks), and Python for Lambda.
